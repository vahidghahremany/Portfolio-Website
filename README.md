# Portfolio Website (Vue.js)

A modern and fully responsive personal portfolio website built with **Vue 3**.  
This project showcases personal information, resume details, and most importantly demonstrates strong practical skills in **Vue**, **Composition API**, **Vue Router**, and **custom animations** without using UI libraries or CSS frameworks.

---

## ðŸŒŸ Project Purpose

The main goal of this project is to:

- Present personal background, skills, and portfolio works.
- Clearly demonstrate hands-on expertise in **Vue 3** ecosystem.
- Implement advanced UI behavior using:
  - **Composition API**
  - **Custom state management inside components**
  - **Vue Router for multi-page navigation**
  - **Responsive layout techniques without frameworks**
  - **Custom scroll and element-based animations** (no external animation libraries)

This portfolio was intentionally developed **without Bootstrap, Tailwind, GSAP, or pre-made UI libraries**, to show **pure front-end implementation skills**.
